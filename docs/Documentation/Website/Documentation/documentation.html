Code Reference

This page documents the custom R functions used across the BIODYNAMICS vegetation co-occurrence project. Function pages are generated from roxygen2 documentation and should not be edited manually.

Function Categories

Category Role
Community Paleo and modern community extraction, reshaping, interpolation, filtering, and aggregation
Taxonomy GBIF-backed classification, manual correction tables, and taxonomic-resolution routing
Traits Trait extraction, quality control, correction, clustering, and functional-type classification
Abiotic Climate extraction, interpolation, collinearity screening, and scaling
Modelling Model input assembly, {sjSDM} fitting, evaluation, MEMs, and variance decomposition
Prediction Spatial and temporal prediction helpers, prediction grids, and animation outputs
Pipeline Configuration access, target-store readers, progress visualisation, and runner utilities
Visualisation Plot builders, project palettes, presentation helpers, and reporting utilities

The active pipeline entry points live in R/Pipelines/. Reusable pipeline construction blocks live in R/Pipelines/_pipes/. The generated reference below links the lower-level functions used by those pipelines.
