Continuous color scale using ORACLE palette

Description:

     US-spelling alias for `scale_colour_oracle_continuous()`.

Usage:

     scale_color_oracle_continuous(values = get_oracle_continuous_palette(), ...)
     
Value:

     A `ggplot2` scale object.

Examples:

     ## Not run:
     
     ggplot2::ggplot(mtcars, ggplot2::aes(wt, mpg, color = disp)) +
       ggplot2::geom_point() +
       scale_color_oracle_continuous()
     ## End(Not run)
