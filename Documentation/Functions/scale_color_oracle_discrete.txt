Discrete color scale using ORACLE palette

Description:

     US-spelling alias for `scale_colour_oracle_discrete()`.

Usage:

     scale_color_oracle_discrete(values = get_oracle_discrete_palette(), ...)
     
Value:

     A `ggplot2` scale object.

Examples:

     ## Not run:
     
     ggplot2::ggplot(mtcars, ggplot2::aes(wt, mpg, color = factor(cyl))) +
       ggplot2::geom_point() +
       scale_color_oracle_discrete()
     ## End(Not run)
