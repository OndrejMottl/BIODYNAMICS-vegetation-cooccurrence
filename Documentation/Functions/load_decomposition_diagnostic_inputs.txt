Load Decomposition Diagnostic Inputs

Description:

     Loads upstream targets needed by the decomposition diagnostic
     framework.

Usage:

     load_decomposition_diagnostic_inputs(
       store_path,
       resolution_id = "genus",
       tar_read_fn = targets::tar_read_raw
     )
     
Arguments:

store_path: Targets store path.

resolution_id: Taxonomic resolution suffix. Default is `"genus"`.

tar_read_fn: Function used to read targets. Defaults to
          `targets::tar_read_raw()`.

Value:

     Named list with upstream data and configuration objects.
