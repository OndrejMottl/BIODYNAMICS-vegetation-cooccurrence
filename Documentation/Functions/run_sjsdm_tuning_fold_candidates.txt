Run sjSDM Tuning Candidates for One Fold

Description:

     Prepares one fold once, then fits, predicts, and scores every
     regularization candidate while preserving failures as compact
     status rows.

Usage:

     run_sjsdm_tuning_fold_candidates(
       data_candidates = NULL,
       list_fold_context = NULL,
       prepare_fold_function = NULL,
       fit_function = NULL,
       predict_function = NULL,
       seed = 900723L,
       epsilon = 1e-06
     )

Arguments:

data_candidates: Candidate table returned by
          [make_sjsdm_regularization_candidates()].

list_fold_context: Fold metadata returned by
          [make_sjsdm_tuning_fold_context()].

prepare_fold_function, fit_function, predict_function: Injectable fold
          preparation, fit, and prediction functions documented by
          [run_sjsdm_tuning_candidates()].

    seed: Non-negative base integer used to derive candidate fit seeds.

 epsilon: Probability clipping tolerance passed to
          [score_sjsdm_tuning_predictions()].

Value:

     Compact candidate-by-fold tuning table with metrics and structured
     status.
