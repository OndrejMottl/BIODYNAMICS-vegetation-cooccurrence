Apply Decomposition Scale Attributes

Description:

     Applies training-set scale attributes to held-out predictor rows.

Usage:

     apply_decomposition_scale_attributes(
       data_predictors = NULL,
       scale_attributes = NULL
     )
     
Arguments:

data_predictors: Data frame with row names and predictor columns.

scale_attributes: Named list of scale attributes from
          `scale_abiotic_for_fit()` or `scale_spatial_for_fit()`.

Value:

     Data frame with the same row names as `data_predictors` and scaled
     columns ordered as `scale_attributes`.
