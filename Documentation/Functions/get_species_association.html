get_species_associationR Documentation

Compute Species Associations

Description

Computes species associations from a fitted Hmsc model object.

Usage

get_species_association(data_sourse)

Arguments

data_sourse

A list containing a fitted Hmsc model under the 'mod' element. Generally, this is the output of the function get_better_model_based_on_fit()

Value

A matrix of species associations.
