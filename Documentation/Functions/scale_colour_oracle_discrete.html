scale_colour_oracle_discreteR Documentation

Discrete colour scale using the ORACLE palette

Description

Maps discrete data values to ORACLE categorical colours using 'ggplot2::scale_colour_manual()'.

Usage

scale_colour_oracle_discrete(values = get_oracle_discrete_palette(), ...)

Arguments

values

Named character vector of colours. Defaults to 'get_oracle_discrete_palette()'.

...

Additional arguments passed to 'ggplot2::scale_colour_manual()'.

Value

A 'ggplot2' scale object.

Examples

## Not run: 

ggplot2::ggplot(mtcars, ggplot2::aes(wt, mpg, colour = factor(cyl))) +

  ggplot2::geom_point() +

  scale_colour_oracle_discrete()
