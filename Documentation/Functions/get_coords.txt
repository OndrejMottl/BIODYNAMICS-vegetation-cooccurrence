Get Dataset Coordinates

Description:

     Extracts unique coordinates for each dataset, excluding
     gridpoints.

Usage:

     get_coords(data)
     
Arguments:

    data: A data frame containing columns 'dataset_name',
          'dataset_type', 'coord_long', and 'coord_lat'. Rows with
          `dataset_type == "gridpoints"` are excluded.

Value:

     A data frame with unique coordinates for each dataset, with
     `dataset_name` as row names.
