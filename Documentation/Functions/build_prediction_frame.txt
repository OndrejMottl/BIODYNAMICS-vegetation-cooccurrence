Build Prediction Frame

Description:

     Builds one animation frame for the IAVS paleo prediction
     animation, showing a spatial prediction grid optionally overlaid
     with observation points.

Usage:

     build_prediction_frame(
       data_frame,
       age_value,
       value_column,
       subtitle_label,
       fill_label,
       fill_limits,
       list_prediction_grid,
       data_world,
       grid_resolution,
       fill_trans = scales::transform_identity(),
       fill_colors = c("black", "white"),
       data_points = NULL,
       point_color = "red",
       metric_label = NULL,
       vec_palette = NULL,
       font_family = "VT323"
     )
     
Arguments:

data_frame: Tibble with columns `age`, `coord_long`, `coord_lat`, and
          the column named by `value_column`.

age_value: Numeric scalar. Age slice to plot.

value_column: Character scalar. Name of the fill value column in
          `data_frame`.

subtitle_label: Character scalar. Plot subtitle.

fill_label: Character scalar. Legend title.

fill_limits: Numeric vector of length 2. Limits for the fill scale.

list_prediction_grid: Named list with `x_lim` and `y_lim` entries.

data_world: World polygon tibble with columns `long`, `lat`, and
          `group`.

grid_resolution: Numeric scalar. Tile width and height in degrees.

fill_trans: Scale transform passed to
          `ggplot2::scale_fill_gradientn()`.

fill_colors: Character vector of length >= 2. Fill gradient colours.

data_points: Optional tibble with `age`, `coord_long`, `coord_lat` for
          overlaid observation points.

point_color: Character scalar. Colour for observation points.

metric_label: Optional character vector. Metrics appended as a caption.

vec_palette: Optional named character vector of ORACLE colours. If
          `NULL`, colours are read with `get_oracle_palette_values()`.

font_family: Character scalar. Font family for all text elements.

Value:

     A `ggplot` object for one prediction frame.
