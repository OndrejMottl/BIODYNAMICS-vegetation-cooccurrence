Assert Prediction Probabilities

Description:

     Validates that a prediction matrix or data frame contains finite
     probability-scale values bounded by zero and one.

Usage:

     assert_prediction_probabilities(data_predictions, tolerance = 1e-08)
     
Arguments:

data_predictions: Numeric matrix or data frame of predicted
          probabilities.

tolerance: Numeric scalar tolerance for small floating-point excursions
          beyond zero and one.

Value:

     The input `data_predictions`, unchanged.

Examples:

     assert_prediction_probabilities(
       data_predictions = matrix(c(0.1, 0.9), nrow = 1)
     )
