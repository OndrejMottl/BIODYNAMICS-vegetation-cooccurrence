prepare_decomposition_fold_inputR Documentation

Prepare Decomposition Fold Input

Description

Rebuilds model input for one route and one train/test split using fold-local response filtering and train-only predictor scaling.

Usage

prepare_decomposition_fold_input(
  route = NULL,
  inputs = NULL,
  train_ids = NULL,
  test_ids = NULL
)

Arguments

route

One-row route data frame or named list.

inputs

List returned by 'load_decomposition_diagnostic_inputs()'.

train_ids

Character vector of training sample row names.

test_ids

Character vector of test sample row names.

Details

This route-specific wrapper obtains the decomposition samples and spatial predictors, then delegates common response filtering, alignment, and predictor scaling to 'prepare_model_fold_input()'.

Value

Named list returned by 'prepare_model_fold_input()', including training input, test input, test observations, a taxon mapping, and diagnostics.

Examples

## Not run:
data_fold <-
  prepare_decomposition_fold_input(
    route = data_route,
    inputs = list_decomposition_inputs,
    train_ids = vec_train_ids,
    test_ids = vec_test_ids
  )

## End(Not run)
