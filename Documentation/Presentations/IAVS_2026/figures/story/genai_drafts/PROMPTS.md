# GenAI Prompt Log

This file records the exact prompts used to generate draft GenAI story assets for the IAVS 2026 ORACLE presentation.
The canonical committed story assets are still generated by `R/generate_story_figures.R` unless this register is updated.

## Note

Through the project development, the terminal was originally named MOTHER, but the final presentation will refer to it as ORACLE.

## Prompt 1: Rejected ORACLE Companion Portrait

```text
Use case: stylized-concept
Asset type: conference presentation atmospheric portrait, saved as assets/mother_face.png
Primary request: create a companion portrait for the ORACLE/MOTHER ecological biosurveillance terminal.
Subject: an androgynous female-presenting synthetic intelligence face, calm and watchful, with subtle human warmth but clearly terminal-like and archival rather than glamorous.
Scene/backdrop: black retro-futuristic command terminal darkness with faint scanlines and soft phosphor reflections.
Style/medium: cinematic sci-fi concept art, realistic but lightly stylized, not a direct reference to any existing film character.
Composition/framing: square portrait, centered head and shoulders, enough negative space around the face for cropping in slides.
Lighting/mood: low-key monitor glow, phosphor green key light, faint cyan rim light, restrained amber highlights, quiet biosurveillance mood.
Color palette: near-black background, #8DF59A phosphor green, #48C7B8 cyan, #FFB35C amber, muted green shadows.
Materials/textures: subtle CRT scanlines, terminal glass reflections, organic micro-textures suggesting vegetation data streams.
Constraints: no text, no logo, no watermark, no horror, no gore, no weapons, no imitation of a specific copyrighted character or actor.
Avoid: purple neon, beige palette, glossy cyberpunk excess, readable interface text, distorted face, extra people.
```

## Prompt 3: ORACLE Terminal Face

```text
Use case: stylized-concept
Asset type: GenAI draft for IAVS 2026 MOTHER companion screen, rough CRT terminal style
Primary request: create a low-fidelity 1979 ship-computer CRT terminal screen for an ecological biosurveillance system called MOTHER.
Subject: a barely human female-presenting face implied only by green phosphor dots and vector scan traces, like a diagnostic point cloud on an old terminal, not a realistic portrait.
Scene/backdrop: black CRT screen with thin green terminal frame, sparse system text blocks, scanlines, raster noise, and dim phosphor bloom.
Style/medium: degraded monochrome green computer terminal output, 1970s spacecraft computer display, technical instrument readout, not cinematic art.
Composition/framing: 16:9 screen capture, face silhouette on the right, MOTHER status text panel on the left, lots of black space.
Lighting/mood: emitted phosphor only, flat screen glow, no dramatic studio lighting.
Color palette: black, phosphor green, dim green-gray, tiny amber status accents only.
Text: no readable text except simple terminal-like fragments such as "MOTHER>" and "SYSTEM ONLINE" if they appear cleanly.
Constraints: make it look old, rough, flat, and terminal-native; use visible scanlines, pixel noise, thin vector lines, and simple panel borders.
Avoid: photoreal face, glossy android, modern hologram, cinematic rim light, smooth dashboard UI, neon cyberpunk, purple, blue glow, polished concept art, detailed typography, logos, watermark.
```

Color palette: black, phosphor green, dim green-gray, tiny amber status accents only.

Constraints: old, rough, pixelated, terminal-native, visibly scanlined; vegetation must be recognizable as trees and shrubs but still rendered as crude CRT data.
